Personal Projects

Clamour.io
Summer/Fall 2017 (continuing development)
Clamour.io, my largest project, is an app created out of interest in the cryptocurrency world. Its purpose is to monitor the cryptocurrency exchange Bittrex.com, continuously collect and analyze collected data, and make suggestions on potentially profitable trading opportunities. Suggestions are presented in a user interface that updates in real time. Users have the ability to save a suggested coin so that its performance can be checked upon at a later date. Future plans include database integration for user logins, an admin dashboard with controllable settings, and a long-term goal of trade integration.
